* Circuito para análise de tempo de descarga do varactor
* Fontes
Vp vp 0 DC 5.94
Vin in 0 PULSE(0 0.1 0 1n 1n 5u 10u)
* Componentes
R1 in out 10k
D1 0 out SMV1493
Vbias out vp 0
* Modelo do diodo varactor
.model SMV1493 D (IS=1e-14 RS=0.1 N=1.024 CJ0=2.8e-11 VJ=0.63 M=0.47 BV=100 IBV=1e-3)
* Análise
.tran 0.1u 50u
* Controle
.control
set color0=white
set color1=black
set xbrushwidth=2
run
plot v(out) v(in)
wrdata varactor_discharge.txt time v(out) v(in)
.endc
.end
